# Required

* node >= 12
* npm >= 6
* make >= 3

# Installation and start command

* `make setup` install utils
* `gendiff -f plain file1.json file2.json` start differ from files file1.json and file2.json

# Command

* `make test` start test 
* `make lint` start lint
